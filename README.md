 酷我音乐搜索方法：
 
 1：直接输入歌曲名称（例如：爱在西元前）
 
 2：输入歌手文字及姓名（例如：歌手周杰伦）
 
 3：输入专辑文字及名称（例如：专辑叶惠美、周杰伦专辑）
